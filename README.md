## ğŸ“± About the Project

Achieving health and fitness goals requires consistent tracking. **MacroLite** cuts through the noise of complex diet apps by focusing strictly on the data that matters. It allows users to log meals instantly, set daily nutritional goals, and visualize their progress with intuitive charts.

### Why MacroLite?
* **Performance First:** Built with Flutter for smooth, native performance on both iOS and Android.
* **Offline-First:** All data is stored locally using **Hive**, ensuring total privacy and instant access without an internet connection.
* **Modern State Management:** Utilizes **Riverpod** for robust and testable state management.
